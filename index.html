Sobre o html

html é uma linguagem de marcação hipertexto(e nao de programação seu burro) que consiste em construir paginas web que vemos e usamos hoje em dia.

Dentro dele temos varios elementos e funcoes que sao a estrutura do html, podemos colocar qualquer coisa como imagens, textos, audios, videos e muito mais.




Estrutura HTML

HTML funciona com etiquetas que se chamam de tags, cada uma é importante para criação de um site.


Doctype Html

Esse elemento não é uma tag mas sim uma declaração, ele é a primeira coisa a ser colocada quando tiver mexendo com HTML.
Consiste em renderizar o html nos navegadores assim não tendo problemas de respostas.
